Python aplicado a datos para Marketing Digital

Esta guía organiza los temas esenciales de Python orientados al análisis de datos, pensada para estudiantes de marketing sin experiencia previa en programación. Cada bloque incluye conceptos clave, ejemplos de código y enlaces a recursos para profundizar.

01

Sintaxis básica de Python

Nivel: básico

Antes de trabajar con datos, necesitas entender cómo "habla" Python: variables, tipos de datos y operaciones simples.

Variables y tipos

int, float, str, bool. Cómo asignar y reconocer tipos de datos.

Operadores

Aritméticos (+, -, *, /), comparación (==, >, <) y lógicos (and, or, not).

Strings

Formateo de texto, f-strings, métodos como .lower(), .strip(), .split().

# Variables y tipos
nombre_campana = "Verano 2026"
presupuesto = 1500.50
activa = True

# f-strings: forma moderna de combinar texto y variables
print(f"La campaña '{nombre_campana}' tiene un presupuesto de S/ {presupuesto}")

# Operaciones con strings
canal = "  Facebook Ads  "
print(canal.strip().lower())  # "facebook ads"
Tip: No necesitas memorizar todo. Lo importante es reconocer estos elementos cuando los veas en código generado por Claude, para poder ajustarlos.

02

Estructuras de datos

Nivel: básico

Las listas y diccionarios son la base sobre la que se construyen los DataFrames. Entenderlas facilita todo lo demás.

Listas

Colecciones ordenadas: [10, 20, 30]. Acceso por índice, slicing.

Diccionarios

Pares clave-valor: {"canal": "Facebook", "gasto": 500}.

Listas de diccionarios

La estructura típica que se convierte en una tabla (DataFrame).

# Lista de impresiones por día
impresiones = [1200, 1500, 980, 2100, 1750]

print(impresiones[0])     # primer elemento: 1200
print(impresiones[-1])    # último elemento: 1750
print(impresiones[1:3])   # slicing: [1500, 980]

# Diccionario: una campaña
campana = {
    "nombre": "Lanzamiento Producto X",
    "canal": "Instagram",
    "gasto": 850.0,
    "conversiones": 42
}

print(campana["canal"])  # "Instagram"

# Lista de diccionarios -> esto se convierte directo en un DataFrame
campanas = [
    {"canal": "Facebook", "gasto": 500},
    {"canal": "Google Ads", "gasto": 1200},
    {"canal": "TikTok", "gasto": 900}
]
Conexión clave: Una lista de diccionarios con la misma estructura es exactamente lo que pd.DataFrame() convierte en una tabla.
03

Control de flujo y funciones

Nivel: básico

Condicionales, bucles y funciones permiten automatizar decisiones y repetir tareas, base de cualquier script de análisis.

# Condicional: clasificar el desempeño de una campaña
ctr = 3.5

if ctr >= 3:
    desempeno = "Alto"
elif ctr >= 1.5:
    desempeno = "Medio"
else:
    desempeno = "Bajo"

print(f"Desempeño: {desempeno}")

# Bucle: recorrer una lista de canales
canales = ["Facebook", "Google Ads", "TikTok"]
for canal in canales:
    print(f"Analizando canal: {canal}")

# Función: reutilizar lógica
def calcular_ctr(clics, impresiones):
    return round(clics / impresiones * 100, 2)

print(calcular_ctr(450, 15000))  # 3.0
Tip: En la práctica, pandas reemplaza la mayoría de bucles manuales (es más rápido y limpio). Pero entender bucles y funciones te ayuda a leer y modificar código generado por IA.
04

NumPy: la base numérica

Nivel: básico-intermedio

NumPy permite hacer cálculos rápidos sobre arreglos de números, y es la base sobre la que está construido pandas.

import numpy as np

# Array de gastos diarios
gastos = np.array([120, 95, 200, 175, 60])

print(gastos.mean())   # promedio
print(gastos.sum())    # suma total
print(gastos.max())    # máximo
print(gastos * 1.18)   # aplicar IGV a todos los valores a la vez
Por qué importa: No usarás NumPy directamente todo el tiempo, pero pandas se basa en él. Entender que las operaciones se aplican a "toda la columna a la vez" (sin bucles) es la idea central.

Pandas: Series y DataFrames

Nivel: intermedio

El corazón del análisis de datos en Python. Un DataFrame es una tabla; una Serie es una columna.

Crear DataFrames

Desde diccionarios, listas o archivos CSV/Excel.

Selección

df['columna'], df[['col1','col2']], .iloc, .loc

Filtrado

Condiciones booleanas: df[df['gasto'] > 500]

import pandas as pd

df = pd.read_csv('campanas.csv')

# Explorar
df.head()
df.shape
df.describe()

df['canal']

# Crear columnas calculadas
df['CTR'] = (df['clics'] / df['impresiones'] * 100).round(2)

06

Limpieza de datos

Nivel: intermedio

Los datos reales casi nunca llegan listos. Esta es una de las habilidades más valoradas en cualquier análisis.

  • Detectar y tratar valores nulos (isnull(), fillna(), dropna())
  • Eliminar duplicados (drop_duplicates())
  • Estandarizar texto (mayúsculas/minúsculas, espacios)
  • Convertir tipos de datos (texto a fecha, texto a número)
  • Detectar valores atípicos (outliers)
# Valores nulos
df.isnull().sum()
df['ingresos'] = df['ingresos'].fillna(0)

# Duplicados
df = df.drop_duplicates(subset=['email'])

# Estandarizar texto
df['nombre'] = df['nombre'].str.lower().str.strip()

# Convertir a fecha
df['fecha'] = pd.to_datetime(df['fecha'])
Tip: Antes de analizar cualquier dataset, dedica tiempo a "inspeccionarlo". Un análisis sobre datos sucios da conclusiones equivocadas, sin importar cuán bonito sea el gráfico final.

Transformación y agrupación

Nivel: intermedio

Aquí es donde los datos "hablan": resumir, comparar y cruzar información para responder preguntas de negocio.

# Agrupar y resumir
df.groupby('canal')['gasto'].sum()

df.groupby('canal').agg(
    gasto_total=('gasto', 'sum'),
    conversiones=('conversiones', 'sum'),
    ctr_promedio=('CTR', 'mean')
).round(2)

# Ordenar
df.sort_values('CTR', ascending=False)

# Combinar tablas
df_completo = df.merge(info_productos, on='producto', how='left')

# Tablas pivote (resumen tipo Excel)
pd.pivot_table(df, values='gasto', index='canal', columns='mes', aggfunc='sum')
Pregunta guía: Cada vez que uses groupby, pregúntate: "¿qué pregunta de negocio estoy respondiendo?" (ej. "¿qué canal generó más conversiones por mes?")
08

Visualización de datos

Nivel: intermedio

Un buen gráfico comunica en segundos lo que una tabla tarda minutos en explicar.

Matplotlib

La librería base. Gráficos de barras, líneas, dispersión.

Seaborn

Construido sobre Matplotlib, gráficos estadísticos más estilizados.

Plotly

Gráficos interactivos (zoom, hover) ideales para dashboards.

import matplotlib.pyplot as plt

# Gráfico de barras
plt.bar(df['canal'], df['CTR'])
plt.title('CTR por canal')
plt.xlabel('Canal')
plt.ylabel('CTR (%)')
plt.show()

# Gráfico de líneas (evolución en el tiempo)
plt.plot(df['fecha'], df['gasto'])
plt.title('Evolución del gasto')
plt.show()
Regla de oro: Elige el tipo de gráfico según la pregunta — barras para comparar categorías, líneas para evolución en el tiempo, dispersión para relaciones entre dos variables.
09

Estadística descriptiva

Nivel: intermedio

Conceptos estadísticos básicos que te ayudan a interpretar resultados correctamente, no solo calcularlos.

  • Medidas de tendencia central: media, mediana, moda
  • Medidas de dispersión: desviación estándar, rango
  • Correlación entre variables (df.corr())
  • Distribución de datos (histogramas)
  • Diferencia entre correlación y causalidad
# Estadísticas descriptivas completas
df.describe()

# Correlación entre variables numéricas
df[['gasto', 'clics', 'conversiones']].corr()

# Histograma de distribución
df['CTR'].hist(bins=10)
plt.title('Distribución del CTR')
plt.show()
Cuidado: Que dos variables estén correlacionadas no significa que una cause la otra. Útil para detectar patrones, no para afirmar relaciones causales sin más análisis.

Introducción a Machine Learning

Nivel: avanzado (opcional)

No es obligatorio para empezar, pero útil para entender hacia dónde puede crecer tu análisis: predicción y segmentación.

Clustering (K-Means)

Agrupar clientes por comportamiento similar (segmentación).

Regresión

Predecir un valor numérico (ej. ventas futuras) a partir de variables.

Clasificación

Predecir una categoría (ej. ¿este lead va a comprar o no?).

from sklearn.cluster import KMeans

# Segmentar clientes según frecuencia y monto de compra
X = df[['frecuencia_compra', 'monto_total']]

modelo = KMeans(n_clusters=3, random_state=42)
df['segmento'] = modelo.fit_predict(X)

df.groupby('segmento')[['frecuencia_compra', 'monto_total']].mean()
Tip: Para esta etapa, lo más eficiente es describirle a Claude tu dataset y objetivo, y pedir el código de modelado. Tu valor está en interpretar los resultados para el negocio, no en escribir el algoritmo desde cero.
